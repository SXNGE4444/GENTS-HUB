# 🤖 AI Automation Orchestrator

**The first unified platform combining OpenHands, crewAI, AutoGPT, Strix, and MoneyPrinterV2.**

## 🚀 Quick Start

### Installation

```powershell
# Clone the repository
git clone https://github.com/SXNGE4444/GENTS-HUB/AI-Automation-Orchestrator
cd AI-Automation-Orchestrator

# Run setup script
.\scripts\setup.ps1
# List available agents
aiauto list-agents

# Run a single agent
aiauto run --agent openhands --task "Write a Python function that calculates fibonacci"

# Run a multi-agent workflow
aiauto run-workflow --workflow examples/dev_security_workflow.yaml
